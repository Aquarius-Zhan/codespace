# Hello World 游戏 - 快速开始指南

## 🎮 如何开始游戏

1. **打开游戏**
   - 游戏会自动加载主菜单

2. **选择难度**
   - 点击三个难度卡片之一：简单、普通或困难
   - 如果你是新手，建议从简单模式开始

3. **开始冒险**
   - 选择难度后，游戏立即开始
   - 你将面对第一个随机生成的关卡

## 🕹️ 游戏操作

### 键盘控制（推荐桌面端）
- **← 左箭头**：向左移动
- **→ 右箭头**：向右移动  
- **↑ 上箭头** 或 **空格键**：跳跃

### 触摸控制（移动端）
- **点击屏幕左侧**：向左移动
- **点击屏幕右侧**：向右移动
- **点击屏幕中间**：跳跃

## 🎯 游戏目标

1. **到达终点**：每个关卡右上方有一个绿色星星，那是终点
2. **避开陷阱**：红色的字母形状都是陷阱，碰到会失败
3. **收集道具**：金色圆圈是道具，尽量收集
4. **完成10关**：连续通过10个关卡即可获得铭牌成就

## ⚠️ 注意事项

- **掉落死亡**：如果掉出屏幕底部，会立即失败
- **陷阱致命**：碰到任何红色陷阱都会失败
- **可以重试**：失败后可以立即重新开始，关卡会重新生成
- **进度重置**：失败后，已通关数会重置为0

## 💡 新手技巧

1. **观察先行**
   - 不要急于前进
   - 先观察平台和陷阱的位置
   - 规划好跳跃路线

2. **掌握节奏**
   - 跳跃有固定的弧线
   - 练习掌握跳跃的距离和高度
   - 在平台边缘起跳可以跳得更远

3. **利用平台**
   - 棕色的平台是安全的
   - 可以站在平台上休息和观察
   - 有些平台可能会移动（高难度）

4. **收集道具**
   - 道具不是必须的，但能帮助你
   - 如果道具位置危险，可以选择跳过
   - 收集的道具会在通关后显示

5. **多次尝试**
   - 失败是正常的，这是Roguelite游戏的特点
   - 每次失败都是学习的机会
   - 关卡会重新生成，给你新的挑战

## 🏆 成就系统

- **简单铭牌**：完成简单难度的10个关卡
- **普通铭牌**：完成普通难度的10个关卡  
- **困难铭牌**：完成困难难度的10个关卡

所有铭牌都会永久保存，下次访问时仍然存在！

## 🎨 界面说明

### 主菜单
- **难度选择卡片**：点击开始游戏
- **字母道具说明**：查看所有道具效果
- **我的成就**：显示已获得的铭牌

### 游戏界面
- **左上角**：当前关卡进度（如：关卡 3/10）
- **右上角**：已收集道具数量
- **暂停按钮**：点击暂停游戏
- **Canvas画布**：游戏主要区域

### 游戏元素
- **橙色方块**：你的角色
- **棕色平台**：可以站立的平台
- **红色字母**：危险的陷阱
- **金色圆圈**：可收集的道具
- **绿色星星**：关卡终点

## 🔧 常见问题

**Q: 游戏卡顿怎么办？**
A: 尝试关闭其他浏览器标签页，或使用性能更好的浏览器（推荐Chrome）

**Q: 我的成就会丢失吗？**
A: 不会，成就保存在数据库中，除非清除浏览器数据

**Q: 可以暂停游戏吗？**
A: 可以，点击左上角的"暂停"按钮

**Q: 失败后可以从当前关卡继续吗？**
A: 不可以，失败后需要从第1关重新开始，这是Roguelite游戏的特点

**Q: 道具有什么用？**
A: 当前版本道具主要用于收集和展示，未来版本会实现道具效果
