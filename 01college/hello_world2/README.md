# Hello World Roguelike Platformer

一个基于"Hello World"主题的2D平台跳跃游戏，具有Roguelike元素，玩法类似超级马里奥。

## 游戏特色

### 核心机制
- **Roguelike元素**：永久死亡（死亡后从第一关重新开始）
- **程序生成关卡**：每个关卡都是随机生成的
- **渐进难度**：随着关卡增加，敌人和陷阱数量增加
- **AI敌人系统**：智能追踪和攻击行为

### 敌人类型
1. **Bug敌人（橙色）**：巡逻型敌人，接近玩家会追击
2. **Error敌人（深红色）**：射击型敌人，会向玩家发射子弹

### 道具系统
- **Code道具（绿色，标记"C"）**：恢复生命值
- **Debug道具（青色圆形）**：获得临时无敌状态
- **Speed道具（黄色闪电）**：获得速度提升
- **Double Jump道具（紫色，标记"DJ"）**：恢复二段跳能力

### 陷阱系统
- **假平台**：看起来正常，但踩上后会延迟坠落
- **陷阱平台**：深红色平台，触碰会造成伤害
- **隐藏危险**：各种意想不到的障碍

### 视觉效果
- **粒子系统**：爆炸、尾迹效果
- **动态HUD**：实时显示生命值、分数、能力状态
- **酷炫背景**：渐变星空效果
- **视觉反馈**：无敌闪烁、道具漂浮动画

## 操作说明

### 基本控制
- **左右方向键**：移动
- **空格键**：跳跃（支持二段跳）
- **R键**：游戏结束后重新开始
- **ESC键**：退出游戏

### 游戏技巧
1. **二段跳**：在空中再次按空格可以进行二段跳
2. **躲避敌人**：利用平台高度差躲避子弹
3. **收集道具**：优先收集生命道具，在困难关卡前获得增益
4. **识别陷阱**：注意灰色的假平台和深红色的陷阱平台
5. **保持距离**：与Error敌人保持距离以躲避射击

## 运行要求

### 系统要求
- Python 3.x
- Pygame 2.x

### 安装依赖
```bash
pip install pygame
```

### 运行游戏
```bash
python hello_world_roguelike.py
```

## 游戏规则

### 胜利条件
- 到达关卡右侧终点进入下一关
- 尽可能获得高分

### 失败条件
- 生命值耗尽（永久死亡，从头开始）
- 掉出地图

### 计分规则
- 收集道具：+100分
- 完成关卡：+1000分

## 技术特点

### 代码架构
- **面向对象设计**：Player、Enemy、Item、Platform等类
- **状态机AI**：敌人的巡逻、追击、攻击状态
- **粒子系统**：可扩展的视觉效果
- **程序化生成**：算法生成关卡布局

### 性能优化
- **视锥剔除**：只渲染可见区域的对象
- **高效碰撞检测**：矩形碰撞检测
- **内存管理**：及时清理过期对象

## 开发说明

### 文件结构
```
hello_world2/
├── hello_world_roguelike.py  # 主游戏文件
└── README.md                 # 说明文档
```

### 主要类说明
- `Game`: 主游戏类，管理游戏循环
- `Player`: 玩家角色类
- `Enemy`: 敌人基类，包含AI逻辑
- `Projectile`: 子弹类
- `Item`: 道具类
- `Platform`: 平台类
- `LevelGenerator`: 关卡生成器
- `ParticleSystem`: 粒子系统
